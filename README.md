# 생활 속 계산기

정적 배포를 전제로 만든 생활 계산기 허브입니다.

## 포함 기능

- 연봉 실수령액 추정기
- 대출 상환 계산기
- 날짜 차이 계산기
- BMI 계산기
- 계산기별 독립 SEO 상세 페이지
- 서비스 소개, 문의하기, 개인정보처리방침 페이지
- `404.html`, `site.webmanifest`
- 로컬 스토리지 기반 입력값 저장

## 실행

```bash
npm run check
```

정적 파일이라서 아무 호스팅에나 바로 배포할 수 있습니다.

## 배포

- Vercel: 프로젝트 import 후 루트 디렉터리 그대로 배포
- Netlify: `G:\Web_cash` 폴더를 드래그 앤 드롭 배포
- GitHub Pages: 저장소 생성 후 정적 사이트로 게시

## 확장 아이디어

- 계산기별 상세 페이지를 더 추가해 검색 유입 확대
- 문의 폼에 실제 전송 기능 연결
- 인기 계산기 사용 흐름을 기준으로 추가 도구 제작
